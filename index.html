Oracle Protocol
Harvey AI LLC · Autonomous AI Systems

Autonomous multi-agent coding framework with a four-tier hierarchy: Oracle → Manager → Lead → Coder. Each layer operates with phase-gated execution — agents can't advance without presenting evidence of completion. A SQLite-backed code knowledge graph over the entire Python source tree lets agents reason about cross-file dependencies and promote learning across runs. Resumable workflows survive session failures. Resolved three classes of deep reliability bugs: runaway self-recursion in agent spawning, inter-agent communication deadlocks, and venv bootstrap failures — making fully autonomous, unsupervised coding runs reliably production-safe.
